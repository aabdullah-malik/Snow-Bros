# ❄️ Snow Bros — OOP Arcade Game (C++ / SFML)

A fully playable 10-level replica of the classic 1990 arcade game **Snow Bros**, built from scratch in **C++ with SFML** as a semester project for Object-Oriented Programming* at FAST-NUCES Islamabad.

---

## 🎮 About the Game

Two snowmen brothers — **Nick** and **Tom** — must battle through 10 levels of enemies to rescue captured princesses. Throw snowballs to encase enemies, then kick the encased snowball into other foes to trigger chain reactions and rack up points.

- **10 levels** with two boss encounters (Level 5: Mogera, Level 10: Gamakichi)
- **Multiplayer support** — two players on the same machine
- **Enemy hierarchy** — Botom → Flyng Fooga Foog → Tornado, plus color variants and bosses
- **Power-up system** — Speed Boost, Snowball Power, Distance Increase, Balloon Mode
- **In-game shop** — spend gems earned from kills and boss drops
- **Login & leaderboard** — file-based persistence for accounts, progress, and scores
